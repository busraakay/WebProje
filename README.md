# ✏️ Yemek-Tarifleri
 ASP.NET Core 3.1 ile geliştirilmiş yemek tariflerini barındıran bir web sitesidir.   
 - ASP.NET Core 3.1 ile Entity Framework Core
 - Ön yüzünde Boostrap 4 kullanılmıştır.
 - Veritabanı olarak MSSQL kullanılmışır.
